🔬 最大机会窗口：VID-G1 Lottie JSON 生成
Sora/Kling 等模型生成的是 MP4 视频，但设计师/工程师日常使用的是 Lottie JSON（可在 iOS/Android/Web 原生渲染，约 50KB）。
OmniLottie（CVPR 2026，已开源） 是首个能直接生成 Lottie JSON 的 AI 模型。这是目前无竞品占位的稀缺能力。
📄 arXiv: 2603.02138 · 建议先做 PoC 验证，由杨帆评估工程接入成本

⚙️ LLM + CSS Animations 🎯 对标 Framer Motion 示例 ✅ 2天可交付
