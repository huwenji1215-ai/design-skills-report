UI-B1 B端增删改查控制台生成
P0 ✅ 2天可交付
▾
输入产品需求描述，直接输出可运行的 B 端管理后台页面代码（含列表、表单、详情三件套）。
输入
  • 业务实体描述（如：用户管理、订单管理）
  • 字段列表（可选，未提供则 AI 自推断）
  • 交互需求（可选，如：可筛选、可批量操作）
  • 输出格式：React 组件代码 / 纯 HTML
输出
  • 列表页（含分页、搜索、筛选）
  • 新增/编辑表单页（含字段校验）
  • 详情页（可选）
  • 可直接运行的完整代码（含样式）
技术路线
LLM 代码生成 → React + Tailwind CSS + shadcn/ui
质量标准（竞品对标：v0.dev）
维度标准
视觉完整度与 v0.dev 生成质量持平（布局合理、无明显错位）
代码可用性直接粘贴到 React 项目可运行，0 报错
交互完整性列表分页、表单提交、错误提示等基础交互覆盖
响应式最低支持 1024px 宽屏
边界（不做什么）
  • 不生成后端接口（只做前端 Mock 数据）
  • 不支持 Vue/Angular（锁定 React 生态）
  • 不做复杂权限体系（超出单页范围）
★★★★★可行性：高 — 已有成熟技术路线（LLM + shadcn/ui），v0.dev 已验证
UI-B2 B端数据可视化仪表盘生成
P1 ⚠️ 需评审
▾
输入数据结构或业务场景，输出含图表的数据监控/分析仪表盘页面代码。
输入
  • 数据结构描述（JSON schema 或字段列表）
  • 图表需求（可选，未提供则 AI 自推断）
  • 业务场景（如：用户增长、GMV 趋势）
输出
  • 含折线图/柱状图/饼图的仪表盘页面
  • 数据筛选器（时间范围/维度切换）
  • 可直接运行的 React 组件代码
技术路线
LLM 代码生成 → React + ECharts / Recharts，竞品对标：Vercel AI
★★★★可行性：较高 — 图表库成熟，LLM 生成图表代码已验证
UI-A1 单页 HTML 原型生成
P1 ✅ 2天可交付
▾
输入需求描述，快速生成一个可浏览器打开的单页静态 HTML 原型，用于走查和沟通。
输入
  • 功能需求描述（自然语言）
  • 设计风格参考（可选）
  • 关键界面元素（可选）
输出
  • 单一 HTML 文件（内嵌 CSS+JS）
  • 浏览器直接打开可预览
  • 可分享给设计师/PM 走查
技术路线
LLM 直出纯 HTML，竞品对标：Claude Artifacts
★★★★★可行性：高 — 最基础的 LLM 代码生成能力，零外部依赖
UI-B5 UI 图标智能选配
P1 ✅ 2天可交付
▾
输入 UI 功能描述，自动推荐匹配图标，输出 SVG 代码片段，轻量高频。
输入
  • 功能或页面描述（如：「用户个人中心」）
  • 图标风格偏好（可选：线性/填充/双色）
输出
  • 3-5 个候选图标推荐
  • 每个图标的 SVG 代码
  • 使用场景说明
技术路线
语义检索 + SVG 输出，对标：Figma 图标插件
★★★★★可行性：高 — 语义检索成熟，SVG 图标库丰富（Lucide/Heroicons）
UI-B3 H5 营销落地页生成
P1 ✅ 2天可交付
▾
输入品牌信息和活动描述，生成可直接投放的 H5 营销落地页代码（移动端适配）。
输入
  • 活动主题/品牌色/CTA 文案
  • 关键卖点（可选，AI 自推断）
  • 风格参考（可选）
输出
  • 移动端 375px 适配的 H5 单页
  • 含动效（入场/滚动触发）
  • 含 CTA 按钮与表单
技术路线
LLM + 移动端 HTML/CSS，竞品对标：Framer AI
★★★★★可行性：高 — 移动端 HTML 生成成熟，无复杂外部依赖
